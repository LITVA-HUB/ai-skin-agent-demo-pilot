# Conversion Layer Notes

## Goal
Increase desire, clarity, and next-step momentum in the beauty advisor.

## Conversion mechanics to reinforce
- hero product first
- supporting products after anchor
- bundle framing instead of flat lists
- aspirational but concise language
- clear next-step CTA in every response
- premium upsell path and budget fallback path

## UX priorities
- first response should create want fast
- follow-up should reduce choice friction
- product order should feel intentional
- recommendations should feel like a look, not a dump
